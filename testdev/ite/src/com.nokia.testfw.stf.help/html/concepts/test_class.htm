Test classes

In STF, test cases are collected into separate classes, called test class, which contain the actual test case implementation.

There can be several test cases implemented in one test class, and test cases can be defined, for example, by a test case file (a configuration file). In addition, STF supports running different test cases simultaneously from several test classes.

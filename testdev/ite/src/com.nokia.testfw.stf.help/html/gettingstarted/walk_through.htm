Basic walk-through

This is the basic order of tasks when using STF:

  1. start the tool, either by
    • creating a new STF project, or
    • importing an existing STF project into the Carbide.c++ IDE.
  2. Add the test module to the STF configuration file.
  3. Add your test cases to the script file(a file containing STF test cases) (when using the test class).
  4. Using Carbide, run the test cases.
  5. (Optional) Send a test drop to the ATS system to remotely run the STF test cases.

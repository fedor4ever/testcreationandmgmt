STF of ITE 1.0.0

STF(Symbian Test Framework) plug-in is a part of ITE(Integrated Test Environment), which is a set of Carbide extensions that supports multiple test frameworks including STF and SUT(Symbian Unit Test).

Product description

STF is a test harness that is designed for testing Symbian non-UI components. It is a toolkit for test case implementation and test case execution.

STF plug-in is an ITE Carbide.c++ Extension that enables you to create test classes and test cases with ease, as well as configure some of STF's options, such as global settings for the test environment.

To start working with STF plug-in, in Carbide, go to File > New > Other, and in Carbide Extensions > Test Frameworks select Symbian Test Framework.

Features

STF lets you to do the following:

  • Create STF test classes
  • Add new test cases to test classes
  • Edit STF configuration files
  • Edit STF scripts
  • Run test classes on an emulator and on hardware (through the Automatic Test System (ATS) Server)
  • Get product help

System requirements

Basic requirements:

  • Windows 2000, Windows XP
  • Minimum Platform/SDK build PC.
  • Normal Platform/SDK build PC.

Additional requirements:

  • STF needs to be installed on a Symbian platform environment. (With STF, you will also get the atsinterface.exe for remotely running the STF test cases.)
  • ATS Server set up for running test classes on hardware.

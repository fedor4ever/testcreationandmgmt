Creating STF cases

To create an STF case, do the following:

  1. Select a header file for which you want to create test cases.
  2. Right-click the selected header file then choose Test Frameworks > Generate STF cases...
  3. Define the test classes and methods. Refer to step 6 of "Creating STF projects".

Product description

STIF is a test harness that is designed for testing Symbian non-UI components. It is a toolkit for test case implementation and test case execution.

STIF Carbide.c++ Extension enables you to create test modules and test cases with ease, as well as configure some of STIF's options, such as global settings for the test environment.

To start working with STIF, go to File > New > Other, and select STIF Test Module.

Features

  • Creating STIF test modules
  • Adding new test cases to test modules
  • Editing STIF configuration file
  • Editing STIF scripts
  • Running test modules on emulator and hardware (through Automatic Test System (ATS) Server)
  • Product help

Enhancements

  • Fixed: The Outline view of the Script Editor window displayed faulty content.

New features

  • A TestDrop plug-in for running test modules on emulator and hardware.

    TestDrop is a tool for making ATS test drop from STIF plug-in project and sending it to ATS Server or running it on emulator software through the atsinterface.

    You can start TestDrop from the Carbide > Send TestDrop, from the Alt+D shortcut, or by right-clicking on a .cfg file and selecting Send TestDrop.

System requirements

Basic requirements:

  • Windows 2000, Windows XP
  • Minimum Platform/SDK build PC.
  • Normal Platform/SDK build PC.

Additional requirements:

  • STIF needs to be installed on the Symbian OS environment. (With STIF, you will also get the atsinterface.exe for remotely running the STIF test cases.)
  • ATS Server for running test modules on hardware.

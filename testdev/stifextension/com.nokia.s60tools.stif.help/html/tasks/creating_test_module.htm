Creating test modules

To create test modules, do as follows:

  1. Select File > New > Other.
  2. Select Carbide Extensions > STIF test module and click Next.
  3. Select one or more Symbian OS SDKs to be used as the build configuration.
  4. On the Test Module Wizard page, choose the module type. For more information on the types, see Test modules.
  5. Give a name for the new module.
  6. (Optional) To create the test module to the Carbide workspace, select the check box (by default the new module is created to the selected environment to the path specified).
  7. Choose the path where the new module should be created.
  8. Click Finish.

    The test module is created and can be viewed in the Project Explorer.

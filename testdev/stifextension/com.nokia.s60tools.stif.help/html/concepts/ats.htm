Automatic Test System (ATS)

Automatic Test System (ATS) provides an automated testing environment for Symbian OS devices. ATS supports booting the devices, flashing test images to them, copying the required files to devices, and executing the defined tests automatically in the target devices. Test execution results are also available from the ATS.

ATS Server is a PC where ATS is installed. The ATS Server is aware of the whole ATS3 network.

ATS Test Drop (or a test package) is a .zip file that contains the test.xml file, test binaries, test script file, and flash images for the target device if it is defined.

The ATS Server connection is not needed for running tests in an emulator environment. The atsserverinterface.exe is used for this purpose and it is delivered as part of STIF.
